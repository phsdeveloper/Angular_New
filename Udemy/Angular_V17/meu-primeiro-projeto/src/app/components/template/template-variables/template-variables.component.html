29. Template Variables

Template Variables nos perimte acessar

  • Obter dados dos atributos de um elemento HTML como por exemplo: Class, id, value, type, etc
  • Obter dados de outro componente, mas do typeScript (pelo menos na aula foi mostrado apenas essa opção)

Templete variables permite ter acesso a todas as propriedades de um elemento HTML como se estivessem vindo do type script {{ <elemento > } } (Text Interpolation) como os exemplos a seguir mas ele não é reativo, ou seja, só pode ser capturado quando a tela é carregada. A sintaxe para isso é #Identificador onde o Identificador pode ser qualquer nome exceto quando .

Dados vindo de outro componente como por exemplo o template-binding da aula anterior

Para que os dados sejam carregados de outro componente é necessário:

  • Importar o componente no arquivo TypeScript imports: [NewComponent]
  • Ter importado no TypeScript as bibliotecas ElementRef e ViewChild sintaxe: {{NewComponent_ImportText}}
  • No TypeScript criar uma variavel com a seguinte sintaxe: {{NewComponent_CodigoTS}}
  • Referenciar o componente no HTML, sem isso não funciona {{NewComponent_tagHTML}}

Com isso é possível acessar não só o HTML do componente mas como as funções e propriedades/variaveis do arquivo typeScript
