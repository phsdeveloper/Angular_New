    Dados que podemos extrair em um for de uma lista
      

    Uma das maiores diferencas entre o Angular na versao antiga e da nova é usar $ antes das propriedades default

    Este exemplo mostra como usar a diretiva *ngFor do Angular para iterar sobre uma lista de itens. O trecho de código contém várias variáveis de contexto que fornecem informações sobre o loop.

                                    
                                    {{'@for (item of items; track item.name;'}}
                                        let i = $index;
                                        let c = $count;
                                        let f = $first;
                                        let l = $last;
                                        let e = $even;
                                        let o = $odd;

    Explicação das Variáveis de Contexto

    • item of items: Itera sobre a lista de itens.
    • track item.name: Usa a propriedade name de cada item para otimizar o rastreamento de itens no loop.
    • let i = $index: Armazena o índice atual do loop na variável i.
    • let c = $count: Armazena o número total de itens na variável c.
    • let f = $first: Armazena um valor booleano que é true se o item atual for o primeiro na lista.
    • let l = $last: Armazena um valor booleano que é true se o item atual for o último na lista.
    • let e = $even: Armazena um valor booleano que é true se o índice do item atual for par.
    • let o = $odd: Armazena um valor booleano que é true se o índice do item atual for ímpar.
