Versão mais atual do Angular

Na versão mais nova do angula é necessário apenas colocar um {{'@if(Condicao booleana) { } @else { }'}}, bem parecido com MVC

@if (bl_isTrue) {
O valor da variavel bl_isTrue = {{bl_isTrue}}
} @else {
O valor da variavel bl_isTrue = {{bl_isTrue}}
}
Versão mais antiga do Angular

Nas versões mais antigas é necessário adicionar no elemento alvo a notação {{'*ngIf="Condicao Booleana"'}}, ou utilizar um ng-container com um ng-template
